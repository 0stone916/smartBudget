# SmartBudget - 가계부 프로젝트

간단한 개인 가계부 관리 웹 애플리케이션입니다.  
React + MyBatis + Spring Boot + JWT 기반 인증 구조로 구현했습니다.

---

## 주요 기능

- 사용자 회원가입 / 로그인
- JWT 기반 인증 및 사용자별 데이터 분리
- 월별 예산 등록/조회/수정/삭제
- 카테고리별 지출 등록/조회/수정/삭제
- 예산 대비 지출 Pie Chart 시각화
- 지출 초과 시 경고 표시
- Axios 인터셉터를 통한 JWT 자동 첨부

---

## 사용 기술

- Frontend: React, Axios
- Backend: Spring Boot, MyBatis
- Database: MySQL
- 인증: JWT
- 패스워드 암호화: BCryptPasswordEncoder
- 차트: react-chartjs-2, Chart.js
